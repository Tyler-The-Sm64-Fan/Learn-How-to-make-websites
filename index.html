HTML (index.html)

HTML (HyperText Markup Language) is the structure of every website. The main file of a website is called index.html. When someone visits your site, this file loads first.

Basic HTML example:


<!DOCTYPE html>

<html>

<head>

    <title>My Website</title>

</head>

<body>

    <h1>Hello World</h1>

    <p>This is my website</p>

</body>

</html>

CSS (style.css)

CSS (Cascading Style Sheets) controls how the website looks. It is usually stored in a file named style.css.

Basic CSS example:


body {

    background-color: lightgray;

    font-family: Arial;

}

h1 {

    color: blue;

}

CSS is connected to HTML using: <link rel="stylesheet" href="style.css">

JavaScript (script.js)

JavaScript makes websites interactive. It can react to clicks, change text, and control behavior. JavaScript is usually saved as script.js.

Basic JavaScript example:


function sayHello() {

    alert("Hello from JavaScript!");

}

GitHub Pages

GitHub Pages allows you to host websites for free. You upload your index.html, style.css, and script.js files to a GitHub repository.

Your website will be available at:
https://username.github.io/repository-name/
